Setting up Git

  1. Download and install the latest version of Git.
  2. On your computer, open your command line application.

  3. Tell Git your name so your commits will be properly labeled. Type everything after the $ here: 

    git config --global user.name "YOUR NAME"

  4. Tell Git the email address that will be associated with your Git commits. The email you specify should be the same one you used to sign up for GitHub. 

    git config --global user.email "YOUR EMAIL ADDRESS"
